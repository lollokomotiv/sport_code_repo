## xA — Expected Assists

Requires `--score-xa`. Uses three models together:
- `xa_pshot_360.joblib` → P(pass leads to shot)
- `xg_model_360_no_penalty.joblib` → xG of the resulting shot (key passes)
- `xa_zone_xg_lookup.joblib` → avg xG by pitch zone (fallback for non-key passes)

### Single match
```bash
python score_match.py \
  --match-id 3788741 \
  --data-root /path/to/open-data/data \
  --model-name xg_model_360_no_penalty \
  --score-xa
```

### Single match + assist map
```bash
python score_match.py \
  --match-id 3788741 \
  --data-root /path/to/open-data/data \
  --model-name xg_model_360_no_penalty \
  --score-xa \
  --plot-assist-map \
  --assist-map-output assist_map.png
```

### xG + xA together
```bash
python score_match.py \
  --match-id 3788741 \
  --data-root /path/to/open-data/data \
  --model-name xg_model_360_no_penalty \
  --shot-scope all_non_penalty \
  --require-360 \
  --plot \
  --score-xa \
  --plot-assist-map
```

## Python API

```python
from score_match import (
    score_match, summarize, plot_shot_map,
    score_xa_match, summarize_xa, plot_assist_map,
)
from pathlib import Path

DATA_ROOT  = Path("/path/to/open-data/data")
MODELS_DIR = Path("../models")

# xG
shots = score_match(
    model_path=MODELS_DIR / "xg_model_360_no_penalty.joblib",
    data_root=DATA_ROOT,
    match_id="3788741",
    require_360=True,
    shot_scope="all_non_penalty",
)
print(summarize(shots))
plot_shot_map(shots, title="Match 3788741")

# xA
passes = score_xa_match(
    pshot_model_path=MODELS_DIR / "xa_pshot_360.joblib",
    xg_model_path=MODELS_DIR   / "xg_model_360_no_penalty.joblib",
    zone_xg_path=MODELS_DIR    / "xa_zone_xg_lookup.joblib",
    data_root=DATA_ROOT,
    match_id="3788741",
)
print(summarize_xa(passes))
plot_assist_map(passes, title="Match 3788741")
```

## Output columns

**xG** (`score_match`):

| Column | Description |
|---|---|
| `team`, `player` | Team and player name |
| `shot_x`, `shot_y` | Shot location |
| `xg` | Predicted xG (0–1) |
| `goal` | 1 if goal, 0 otherwise |

**xA** (`score_xa_match`):

| Column | Description |
|---|---|
| `team`, `player` | Team and player name |
| `pass_start_x/y` | Pass origin |
| `pass_end_x/y` | Receiver position |
| `p_shot` | P(pass → shot) from P_shot model |
| `shot_xg` | xG from xG model (key passes only) |
| `zone_xg` | Zone avg xG (fallback) |
| `xa` | Final xA = p_shot × xg_used |
| `shot_assist` | True if pass directly created a shot |
| `goal_assist` | True if pass led to a goal |
